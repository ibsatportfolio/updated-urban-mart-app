# updated-urban-mart-app
#### An app for online ordering of fresh fruits and vegetables.
- Implemented GetX as a state management solution in the application, leveraging its reactive approach, dependency injection, and powerful navigation system.
- For streamlined development, improved code organization, and enhanced application performance."
- Used Retrofit's annotation-based approach to define API service methods, automatically serializing request parameters and headers.
- For seamlessly handling the network requests and deserializing the JSON responses into Dart model classes.
